# Урок 17 - Семантические теги HTML5

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по семантическим тегам HTML5. Изучение `<header>`, `<footer>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<figure>`, `<mark>`, `<time>`.

---

## 🚀 Основные возможности

- **`<header>`**: Заголовок страницы или раздела
- **`<footer>`**: Подвал страницы или раздела
- **`<nav>`**: Навигационное меню
- **`<main>`**: Основное содержимое страницы
- **`<article>`**: Независимый контент (статья, пост)
- **`<section>`**: Раздел документа
- **`<aside>`**: Боковая панель
- **`<figure>` и `<figcaption>`**: Изображение с подписью
- **`<mark>`**: Выделенный текст
- **`<time>`**: Время и дата

---

## 🛠 Технологический стек

- **HTML5**: Семантические теги
- **CSS3**: Стилизация семантических блоков
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
