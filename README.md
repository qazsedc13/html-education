# Сайт Атлант (HTML Education)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию сайта компании "Атлант" в рамках изучения HTML и CSS. Практическая работа по вёрстке корпоративного веб-сайта с каталогом продукции.

---

## 🚀 Основные возможности

- **Корпоративный сайт**: Многостраничный сайт компании "Атлант".
- **Каталог продукции**: Страницы с описанием товаров.
- **Навигация**: Меню для перехода между разделами.
- **Контактная информация**: Страница с контактными данными.
- **Адаптивная структура**: Блочная вёрстка с использованием div.

---

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка страниц.
- **CSS3**: Стилизация элементов интерфейса.
- **Изображения**: JPEG/PNG для графики сайта.

---

## 📋 Предварительные требования

- **Веб-браузер**: Современный браузер с поддержкой HTML5.
- **Текстовый редактор**: Notepad++, VS Code, Sublime Text или аналог.
- **ОС**: Любая платформа с веб-браузером.

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout atlant
```

### 3. Открытие сайта
Откройте главную страницу в браузере:
```bash
# Windows
start "АтлантРС.html"

# macOS/Linux
open "АтлантРС.html"
```

## 💡 Примеры использования

### Просмотр сайта
- Откройте `АтлантРС.html` для просмотра главной страницы
- Используйте навигационное меню для перехода по разделам
- Изучите структуру HTML-документа

### Редактирование контента
Откройте файл в текстовом редакторе и измените:
- Заголовки `<h1>`, `<h2>`
- Текстовое содержимое параграфов `<p>`
- Ссылки меню `<a href="...">`

---

## ⚠️ Устранение неполадок

**Страница не отображается:**
Убедитесь, что файл `АтлантРС.html` существует и не повреждён.

**Стили не загружаются:**
Проверьте пути к CSS файлам в директории `АтлантРС_files/`.

**Кодировка неверная:**
Откройте файл в редакторе с кодировкой UTF-8 или Windows-1251.
