# Адаптивный сайт (HTML Education)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию адаптивного сайта в рамках изучения HTML5 и CSS3. Практическая работа по вёрстке сайта с адаптивной версткой.

---

## 🚀 Основные возможности

- **Адаптивная вёрстка**: Сайт подстраивается под разные размеры экранов.
- **Медиа-запросы**: Использование CSS3 media queries для различных устройств.
- **Гибкие изображения**: Изображения масштабируются с помощью `max-width: 100%`.
- **Навигация**: Меню с переходом на главную, регистрацию, вход, "О нас".
- **Поиск**: Поле поиска в шапке сайта.

---

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка страниц.
- **CSS3**: Адаптивные стили, медиа-запросы, flexbox/grid техники.
- **Адаптивный дизайн**: Подход "mobile-first".

---

## 📋 Предварительные требования

- **Веб-браузер**: Современный браузер с поддержкой HTML5 и CSS3.
- **Текстовый редактор**: Notepad++, VS Code, Sublime Text или аналог.
- **ОС**: Любая платформа с веб-браузером.

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout html5-css3/adapt
```

### 3. Открытие сайта
Откройте www/index.html в браузере.
Для тестирования адаптивности измените размер окна браузера.

---

## 💡 Ключевые особенности адаптивности

### Гибкие изображения
```css
img {
    max-width: 100%;
    height: auto;
}
```

### Адаптивные блоки
- Шапка и подвал растягиваются на всю ширину
- Элементы перестраиваются в зависимости от размера экрана
