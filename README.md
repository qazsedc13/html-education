# HTML Education Project

Учебный проект для изучения основ веб-разработки (HTML и CSS).

## Описание
Данный проект представляет собой пример создания простого многостраничного веб-сайта. Он включает в себя структуру с заголовком (header), навигационными ссылками, формами регистрации и входа, а также базовое оформление стилей через внешний CSS-файл. Проект идеально подходит для понимания семантики HTML5 и основ верстки.

## Технологический стек
- **Разметка:** HTML5
- **Стилизация:** CSS3
- **Инструментарий:** Любой текстовый редактор и веб-браузер.

## Установка и запуск

Проект является статическим и не требует сервера.

1. Склонируйте репозиторий:
   ```bash
   git clone git@github.com:qazsedc13/html-education.git
   ```
2. Перейдите в папку `www/`.
3. Откройте файл `index.html` в вашем любимом веб-браузере.

## Примеры использования
- **Структура страницы:** В `index.html` реализован заголовок с логотипом, поисковой строкой и кнопками авторизации.
- **Стилизация:** Файл `css/style.css` содержит правила для оформления элементов хедера, ссылок и текстовых полей.
- **Практика:** Файл `first.html` в корне проекта можно использовать для черновых экспериментов с разметкой.

## Структура проекта
- `www/` — основная директория сайта.
  - `index.html` — главная страница проекта.
  - `css/` — папка с каскадными таблицами стилей (`style.css`).
  - `img/` — директория для изображений и иконок (`favicon.ico`).
- `first.html` — отдельный HTML-файл для дополнительных упражнений.
- `image.jpeg` — пример изображения для вставки на страницы.

## Зависимости и требования
- Любой современный веб-браузер.
- Знание основ HTML-тегов и CSS-селекторов.
