# Создание сайта 19 - Адаптивность

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный урок по адаптивности сайта: медиа-запросы, мобильная версия, резиновый дизайн, retina-изображения.

---

## 🚀 Основные возможности

- **Медиа-запросы**: @media для разных экранов.
- **Mobile-First**: Подход к разработке.
- **Мобильная версия**: Адаптация под телефон.
- **Планшетная версия**: Средние экраны.
- **Retina**: Изображения для высоких DPI.

---

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка.
- **CSS3**: Media Queries, Flexbox, Grid.
- **Веб-браузер**: DevTools для тестирования.

---

## 📋 Предварительные требования

- **Веб-браузер**: Chrome/Firefox с DevTools.
- **Текстовый редактор**: Блокнот или VS Code.

---

## 🚀 Установка и запуск

```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
git checkout sozdanie_sajta-19/adaptivnost_sajta
# Откройте index.html в браузере
# Используйте DevTools (F12) для проверки адаптивности
```
