# Урок CSS 4 - Псевдоклассы

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный урок по псевдоклассам CSS. Изучение :hover, :active, :focus, :visited, :first-child, :last-child, :nth-child(), :not(), :checked, :disabled.

---

## 🚀 Основные возможности

- **Псевдоклассы состояния**: :hover, :active, :focus, :visited
- **Структурные псевдоклассы**: :first-child, :last-child, :nth-child()
- **Псевдоклассы форм**: :checked, :disabled, :required, :valid
- **Псевдокласс :not()**: Отрицание
- **Комбинации**: Использование нескольких псевдоклассов

---

## 🛠 Технологический стек

- **HTML5**: Формы и элементы
- **CSS3**: Псевдоклассы
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
