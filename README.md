# Урок 12 - Тег head и мета-теги

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по разделу `<head>` HTML-страницы. Изучение мета-тегов для SEO, кодировки, viewport, подключения стилей и скриптов, favicon, title.

---

## 🚀 Основные возможности

- **Мета-теги**: description, keywords, author, robots
- **Кодировка**: `<meta charset="UTF-8">`
- **Viewport**: Адаптивность для мобильных устройств
- **Подключение CSS**: Тег `<link rel="stylesheet">`
- **Подключение JS**: Тег `<script>`
- **Favicon**: Иконка сайта
- **Заголовок**: Тег `<title>`

---

## 🛠 Технологический стек

- **HTML5**: Теги head и meta
- **CSS3**: Подключение стилей
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
