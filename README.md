# Урок 4 - HTML-комментарии

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по использованию комментариев в HTML. Изучение синтаксиса `<!-- комментарий -->` для документирования кода и временного отключения участков разметки.

---

## 🚀 Основные возможности

- **Синтаксис комментариев**: `<!-- текст -->`
- **Документирование**: Пояснения в коде
- **Отладка**: Временное скрытие элементов

---

## 🛠 Технологический стек

- **HTML5**: Разметка страниц
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout lesson-4-comment
```

### 3. Открытие сайта
```bash
start example.html
```
