# Урок CSS 1 - Первые стили

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный урок по основам CSS. Изучение подключения стилей, синтаксиса правил, селекторов, свойств и значений, каскадности и наследования.

---

## 🚀 Основные возможности

- **Подключение CSS**: Встроенные, внешние, inline-стили
- **Синтаксис**: Селекторы, свойства, значения
- **Типы селекторов**: Тег, класс, ID, атрибут
- **Каскадность**: Приоритеты стилей
- **Наследование**: Передача стилей дочерним элементам

---

## 🛠 Технологический стек

- **HTML5**: Разметка
- **CSS3**: Стилизация
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout lesson-css-1/perviy
```

### 3. Открытие сайта
```bash
start index.html
```
