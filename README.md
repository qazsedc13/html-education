# HTML5 Видео образование (HTML Education)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по изучению HTML5 и работе с видео. Практические примеры базовой разметки HTML5.

---

## 🚀 Основные возможности

- **HTML5**: Изучение базовой разметки HTML5.
- **Теги HTML**: Основные структурные элементы веб-страницы.
- **Мета-теги**: Настройка кодировки и метаинформации.
- **Тело документа**: Структура и содержимое веб-страницы.
- **Видео**: Подготовка к работе с HTML5 video элементом.

---

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка страниц.
- **UTF-8**: Кодировка документа.

---

## 📋 Предварительные требования

- **Веб-браузер**: Современный браузер с поддержкой HTML5.
- **Текстовый редактор**: Notepad++, VS Code, Sublime Text или аналог.
- **ОС**: Любая платформа с веб-браузером.

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout html5-video-education
```

### 3. Открытие примеров
Откройте любой HTML файл в браузере:
- `first.html` - Первый пример (старый формат)
- `example.html` - Пример проверки
- `Untitled-1.html` - Пустой шаблон

## 💡 Основы HTML5

### Базовая структура документа
```html
<!doctype html>
<html>
<head>
    <meta charset="utf-8">
    <title>Заголовок страницы</title>
</head>
<body>
    Содержимое страницы
</body>
</html>
```

###DOCTYPE
```html
<!doctype html>
```
Объявляет документ как HTML5.
