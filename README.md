# Создание сайта 8 - Футер

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию футера (подвала) сайта в рамках курса по созданию сайта с нуля. Изучение структуры, стилей, контактов и копирайта.

---

## 🚀 Основные возможности

- **Футер**: Создание подвала сайта
- **Контактная информация**: Адрес, телефон, email
- **Копирайт**: Авторские права
- **Социальные сети**: Ссылки на профили
- **CSS-оформление**: Стилизация футера

---

## 🛠 Технологический стек

- **HTML5**: Разметка футера
- **CSS3**: Стилизация
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-8/pishem_futer_dlya_sajta
```

### 3. Открытие сайта
```bash
cd mySite
start index.html
```
