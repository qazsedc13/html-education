# Урок 16 - Формы: select

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по элементу `<select>`. Изучение выпадающих списков, тегов `<option>` и `<optgroup>`, атрибутов multiple, size, selected.

---

## 🚀 Основные возможности

- **Тег `<select>`**: Выпадающий список
- **Тег `<option>`**: Пункты списка
- **Тег `<optgroup>`**: Группировка пунктов
- **Атрибут multiple**: Множественный выбор
- **Атрибут selected**: Предвыбранный пункт

---

## 🛠 Технологический стек

- **HTML5**: Формы и select
- **CSS3**: Оформление списков
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
