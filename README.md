# Создание сайта 11 - Блоки сайта

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию блоков сайта в рамках курса по созданию сайта с нуля. Изучение сетки, колонок, карточек и секций.

---

## 🚀 Основные возможности

- **Сетка**: Блочная структура сайта
- **Колонки**: Многоколоночный макет
- **Карточки**: Блоки контента
- **Секции**: Разделы страницы

---

## 🛠 Технологический стек

- **HTML5**: Разметка блоков
- **CSS3**: Grid/Flexbox верстка
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-11/sozdanie_blokov_sajta
```

### 3. Открытие сайта
```bash
cd mySite
start index.html
```
