# Урок 5 - Текстовые теги HTML

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по основным текстовым тегам HTML. Изучение заголовков `<h1>`-`<h6>`, параграфов `<p>`, переноса строки `<br>`, выделения текста `<strong>`, `<em>`.

---

## 🚀 Основные возможности

- **Заголовки**: Теги `<h1>` - `<h6>` для структурирования текста
- **Параграфы**: Тег `<p>` для абзацев
- **Перенос строки**: Тег `<br>`
- **Выделение текста**: `<strong>`, `<b>`, `<em>`, `<i>`, `<u>`, `<s>`
- **Горизонтальная линия**: Тег `<hr>`

---

## 🛠 Технологический стек

- **HTML5**: Текстовые теги
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout lesson-5-teg-text
```

### 3. Открытие сайта
```bash
start example.html
```
