# Урок 18 - Верхнее меню (Навигация)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный урок по созданию верхнего навигационного меню. Изучение горизонтального меню, выпадающих списков, стилизации ссылок и фиксированной навигации.

---

## 🚀 Основные возможности

- **Горизонтальное меню**: Тег `<nav>` со списком ссылок
- **Выпадающие списки**: Вложенные списки `<ul>`
- **CSS-стилизация**: Оформление активных ссылок и hover-эффекты
- **Фиксированное меню**: position: fixed для закрепления сверху
- **Адаптивность**: Мобильное меню (гамбургер)

---

## 🛠 Технологический стек

- **HTML5**: Теги nav и списки
- **CSS3**: Flexbox, позиционирование
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
