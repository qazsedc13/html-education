# О нас - Страница информации (HTML Education)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию страницы "О нас" в рамках изучения HTML и CSS. Практическая работа по созданию информационной страницы о компании.

---

## 🚀 Основные возможности

- **Страница "О нас"**: Информационная страница о компании.
- **Навигация**: Меню с переходом на главную, регистрацию, вход.
- **Поиск**: Поле поиска в шапке сайта.
- **Футер**: Нижний колонтитул с копирайтом.
- **Адаптивная структура**: Блочная вёрстка с использованием semantic HTML.

---

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка страниц.
- **CSS3**: Стилизация элементов интерфейса.
- **Шаблон**: Готовый макет сайта с навигацией.

---

## 📋 Предварительные требования

- **Веб-браузер**: Современный браузер с поддержкой HTML5.
- **Текстовый редактор**: Notepad++, VS Code, Sublime Text или аналог.
- **ОС**: Любая платформа с веб-браузером.

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout html5-css3/about
```

### 3. Открытие сайта
Откройте www/index.html в браузере.

## 📁 Структура проекта

| Файл / Директория | Описание |
| :--- | :--- |
