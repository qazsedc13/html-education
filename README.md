# Урок 7 - Списки HTML

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по созданию списков в HTML. Изучение нумерованных `<ol>`, маркированных `<ul>`, списков определений `<dl>` и вложенных списков.

---

## 🚀 Основные возможности

- **Маркированные списки**: Тег `<ul>` с элементами `<li>`
- **Нумерованные списки**: Тег `<ol>` с нумерацией
- **Списки определений**: Теги `<dl>`, `<dt>`, `<dd>`
- **Вложенные списки**: Многоуровневая структура
- **Атрибуты списков**: type, start, reversed

---

## 🛠 Технологический стек

- **HTML5**: Теги списков
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
