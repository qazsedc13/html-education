# Персона ДВ - Юридическая компания (HTML Education)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![XHTML](https://img.shields.io/badge/XHTML-1.0-teal)
![CSS](https://img.shields.io/badge/CSS-3-blue)

Учебный проект по изучению веб-разработки на примере сайта юридической компании "Персона ДВ". Практическая работа по анализу и изучению готового корпоративного сайта.

---

## 🚀 Основные возможности

- **Корпоративный сайт**: Многостраничный сайт юридической компании.
- **Каталог услуг**: Информация о допусках СРО, лицензиях МЧС.
- **Joomla CMS**: Сайт построен на Joomla 1.5.
- **Новости**: RSS и Atom ленты.
- **Контактная информация**: Телефоны, адрес, схема проезда.
- **Адаптивная верстка**: Поддержка различных устройств.

---

## 🛠 Технологический стек

- **XHTML 1.0 Transitional**: Разметка страниц.
- **CSS3**: Стилизация (layout, typography, print).
- **jQuery**: JavaScript библиотека для интерактивности.
- **Joomla 1.5**: Система управления контентом.

---

## 📋 Предварительные требования

- **Веб-браузер**: Современный браузер с поддержкой HTML5.
- **Текстовый редактор**: Notepad++, VS Code, Sublime Text или аналог.
- **ОС**: Любая платформа с веб-браузером.

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout persona
```

### 3. Открытие сайта
Откройте `persona.html` в браузере.

## 💡 Особенности сайта

### Структура XHTML
```html
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN">
```

### Подключение стилей
```html
<link rel="stylesheet" href="./persona_files/layout.css">
<link rel="stylesheet" href="./persona_files/styles.css">
```

### jQuery
```html
<script type="text/javascript" src="./persona_files/jquery.min.js"></script>
```
