# Урок CSS 6 - Продвинутый CSS

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный урок по продвинутым техникам CSS. Изучение Flexbox, Grid, анимаций, трансформаций, градиентов и медиа-запросов.

---

## 🚀 Основные возможности

- **Flexbox**: Гибкий контейнер и выравнивание
- **CSS Grid**: Двумерная сетка
- **Анимации**: @keyframes и animation
- **Трансформации**: transform (rotate, scale, translate)
- **Градиенты**: linear-gradient, radial-gradient
- **Медиа-запросы**: @media для адаптивности

---

## 🛠 Технологический стек

- **HTML5**: Разметка
- **CSS3**: Продвинутые техники
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout lesson-CSS-6
```

### 3. Открытие сайта
```bash
start index.html
```
