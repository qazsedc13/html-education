# Урок 2 - HTML5 и CSS3

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный урок по основам HTML5 и CSS3. Изучение базовой структуры страницы, подключения стилей, селекторов, свойств текста и блоков.

---

## 🚀 Основные возможности

- **HTML5-структура**: Doctype, head, body
- **Подключение CSS**: Внешние стили
- **Основные селекторы**: Тег, класс, ID
- **Свойства текста**: font-family, font-size, color
- **Свойства блоков**: margin, padding, border

---

## 🛠 Технологический стек

- **HTML5**: Базовая разметка
- **CSS3**: Основы стилизации
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout lesson2/html5-css3
```

### 3. Открытие сайта
```bash
cd www
start index.html
```
