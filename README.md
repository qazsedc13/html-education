# Урок 14 - Формы: textarea

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по элементу `<textarea>`. Изучение многострочного текстового поля, атрибутов rows, cols, placeholder, maxlength, resize.

---

## 🚀 Основные возможности

- **Тег `<textarea>`**: Многострочное текстовое поле
- **Атрибуты размера**: rows (строки), cols (столбцы)
- **Placeholder**: Подсказка внутри поля
- **Maxlength**: Ограничение количества символов
- **Resize**: Управление изменением размера

---

## 🛠 Технологический стек

- **HTML5**: Формы и textarea
- **CSS3**: Оформление текстовых полей
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
