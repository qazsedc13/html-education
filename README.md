# Урок 10 - Таблицы HTML

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по созданию таблиц в HTML. Изучение тегов `<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<tr>`, `<th>`, `<td>`, атрибутов colspan, rowspan, border.

---

## 🚀 Основные возможности

- **Тег `<table>`**: Создание таблицы
- **Строки и ячейки**: Теги `<tr>`, `<th>`, `<td>`
- **Заголовок таблицы**: Тег `<thead>` и `<th>`
- **Тело и подвал**: Теги `<tbody>`, `<tfoot>`
- **Объединение ячеек**: Атрибуты colspan и rowspan
- **Границы таблицы**: Атрибут border и CSS-оформление

---

## 🛠 Технологический стек

- **HTML5**: Теги таблиц
- **CSS3**: Оформление таблиц
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
