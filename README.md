# Создание сайта 12 - Страница "О нас"

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию страницы "О нас" в рамках курса по созданию сайта с нуля. Изучение структуры страницы, команды, истории и миссии компании.

---

## 🚀 Основные возможности

- **Страница "О нас"**: Информация о компании/проекте
- **Команда**: Фото и имена сотрудников
- **История**: Хронология развития компании
- **Миссия**: Цели и ценности компании

---

## 🛠 Технологический стек

- **HTML5**: Разметка страницы
- **CSS3**: Стилизация
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-12/sozdayom_stranichku_o_nas
```

### 3. Открытие сайта
```bash
cd mySite
start index.html
```
