# Создание сайта 5 - Шаблон сайта

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию шаблона сайта в рамках курса по созданию сайта с нуля. Изучение базовой структуры, хедера, контента, футера и подключения стилей.

---

## 🚀 Основные возможности

- **Шаблон сайта**: Базовая структура страницы
- **Хедер**: Заголовок и навигация
- **Контент**: Основная область
- **Футер**: Подвал сайта
- **Подключение шрифтов**: Google Fonts

---

## 🛠 Технологический стек

- **HTML5**: Структура шаблона
- **CSS3**: Стилизация
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-5/shablon_sajta+vsyakie_melochi
```

### 3. Открытие сайта
```bash
cd mySite
start index.html
```
