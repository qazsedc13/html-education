# Создание сайта 15 - Отправка формы (JS + Ajax + PHP)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![PHP](https://img.shields.io/badge/PHP-7.x-blue)

Учебный проект по отправке формы с использованием JavaScript, Ajax и PHP в рамках курса по созданию сайта с нуля. Финальная версия с отправкой данных на сервер.

---

## 🚀 Основные возможности

- **Финальная форма**: Полная отправка данных
- **JavaScript**: Обработка и отправка без перезагрузки
- **PHP**: Получение и сохранение данных
- **Feedback**: Уведомление пользователя о результате

---

## 🛠 Технологический стек

- **HTML5**: Форма и разметка
- **JavaScript**: fetch API / Ajax
- **PHP**: Обработка данных на сервере
- **Веб-сервер**: Apache/PHP

---

## 📋 Предварительные требования

- **Веб-сервер**: OpenServer, Denwer или LAMP
- **PHP 7+**: Для обработки формы
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-15/proverka_polej_JS_+_Ajax_+_PHP_otpravka
```

### 3. Запуск на локальном сервере
```bash

# Запустите локальный сервер с PHP
```
