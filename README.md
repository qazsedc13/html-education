Учебный проект по созданию страниц статей в рамках курса по созданию сайта с нуля. Изучение шаблона статьи, вывода полного текста и навигации.

---

## 🚀 Основные возможности

- **Страница статьи**: Полный текст материала
- **Шаблон**: Единый стиль для всех статей
- **Навигация**: Возврат на главную страницу
- **Мета-теги**: SEO для статьи

---

## 🛠 Технологический стек

- **HTML5**: Шаблон страницы
- **CSS3**: Стилизация
- **PHP**: Динамическая подстановка
- **Веб-сервер**: Apache/PHP

---

## 📋 Предварительные требования

- **Веб-сервер**: OpenServer, Denwer или LAMP
- **PHP 7+**: Для работы с данными
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-17/sozdanie_stranichek_so_statyami
```

### 3. Запуск на локальном сервере
```bash
cd mySite
# Запустите локальный сервер с PHP
```
