# Создание сайта 16 - Вывод статей на главной

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)
![PHP](https://img.shields.io/badge/PHP-7.x-blue)

Учебный проект по выводу статей на главной странице в рамках курса по созданию сайта с нуля. Изучение динамического контента, карточек статей и цикла вывода.

---

## 🚀 Основные возможности

- **Динамический контент**: Вывод из массива/базы данных
- **Карточки статей**: Шаблон статьи
- **Цикл вывода**: Перебор и отображение данных
- **Анонсы**: Краткое описание статей

---

## 🛠 Технологический стек

- **HTML5**: Шаблон страницы
- **CSS3**: Стилизация карточек
- **PHP**: Вывод динамических данных
- **Веб-сервер**: Apache/PHP

---

## 📋 Предварительные требования

- **Веб-сервер**: OpenServer, Denwer или LAMP
- **PHP 7+**: Для работы с данными
- **Текстовый редактор**: Блокнот или VS Code

---
