# Создание сайта 9 - Боковая панель

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию боковой панели (sidebar) в рамках курса по созданию сайта с нуля. Изучение структуры, виджетов, навигации и стилизации боковой панели.

---

## 🚀 Основные возможности

- **Боковая панель**: Создание sidebar для сайта
- **Виджеты**: Поиск, категории, архивы
- **Навигация**: Дополнительное меню в sidebar
- **CSS-стилизация**: Оформление боковой панели
- **Адаптивность**: Скрытие sidebar на мобильных устройствах

---

## 🛠 Технологический стек

- **HTML5**: Разметка боковой панели
- **CSS3**: Flexbox/Grid стилизация
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-9/sozdayom-bokovuyu-panel
```

### 3. Открытие сайта
```bash
cd mySite
# Windows:
start index.html
# macOS:
open index.html
# Linux:
xdg-open index.html
```
