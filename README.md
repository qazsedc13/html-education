# Уроки 8-9 - Ссылки в HTML

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по работе с гиперссылками в HTML. Изучение тега `<a>`, атрибутов href, target, title, якорных ссылок, ссылок на файлы и email.

---

## 🚀 Основные возможности

- **Тег `<a>`**: Создание гиперссылок
- **Атрибут href**: URL назначения ссылки
- **Атрибут target**: Открытие в новом окне/вкладке
- **Якорные ссылки**: Навигация внутри страницы (`#id`)
- **Ссылки на файлы**: Скачивание файлов (атрибут download)
- **Email-ссылки**: Открытие почтового клиента (`mailto:`)

---

## 🛠 Технологический стек

- **HTML5**: Тег ссылки
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
