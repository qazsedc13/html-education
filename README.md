# Создание сайта 7 - Кодировка .htaccess

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![Apache](https://img.shields.io/badge/Apache-.htaccess-red)

Учебный проект по настройке кодировки сайта через файл .htaccess. Изучение UTF-8, редиректов и SEO-настроек для веб-сервера Apache.

---

## 🚀 Основные возможности

- **Файл .htaccess**: Конфигурация Apache
- **Кодировка**: Принудительная установка UTF-8
- **Редиректы**: 301 и 302 перенаправления
- **ЧПУ**: Красивые URL-адреса

---

## 🛠 Технологический стек

- **HTML5**: Разметка страниц
- **Apache**: Конфигурация .htaccess
- **Веб-сервер**: Apache (OpenServer, Denwer, LAMP)

---

## 📋 Предварительные требования

- **Веб-сервер**: Apache (OpenServer, Denwer, LAMP)
- **Текстовый редактор**: Блокнот или VS Code

---
