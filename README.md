# Урок 19 - Специальные теги HTML5

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по специальным тегам HTML5. Изучение `<details>`, `<summary>`, `<meter>`, `<progress>`, `<output>`, `<canvas>`, `<svg>`, `<template>`.

---

## 🚀 Основные возможности

- **`<details>` и `<summary>`**: Раскрываемые блоки
- **`<meter>`**: Визуальный индикатор значения
- **`<progress>`**: Индикатор выполнения
- **`<output>`**: Вывод результата вычислений
- **`<canvas>`**: Рисование на JavaScript
- **`<svg>`**: Векторная графика
- **`<template>`**: Шаблоны контента

---

## 🛠 Технологический стек

- **HTML5**: Специальные теги
- **JavaScript**: Для canvas и динамики
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
