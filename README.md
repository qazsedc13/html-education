# Урок 13 - Теги div и span

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по универсальным тегам `<div>` и `<span>`. Изучение блочной и строчной верстки, создания макета страницы, группировки элементов.

---

## 🚀 Основные возможности

- **Тег `<div>`**: Блочный контейнер для группировки элементов
- **Тег `<span>`**: Строчный контейнер для текста
- **Создание макета**: Верстка страницы с помощью блоков
- **Идентификаторы и классы**: Атрибуты id и class для стилизации

---

## 🛠 Технологический стек

- **HTML5**: Универсальные теги
- **CSS3**: Блочная модель и стилизация
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
