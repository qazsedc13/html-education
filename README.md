# Работа с изображениями (HTML Education)

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по работе с изображениями в HTML5 и CSS3. Практическая работа по добавлению и стилизации изображений на веб-странице.

---

## 🚀 Основные возможности

- **Изображения**: Добавление изображений на веб-страницу с помощью тега `<img>`.
- **Favicon**: Настройка иконки сайта для браузера.
- **Адаптивные изображения**: Масштабирование изображений под разные размеры экранов.
- **Alt-текст**: Описание изображений для доступности и SEO.
- **Навигация**: Меню с переходом между страницами.

---

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка, работа с изображениями.
- **CSS3**: Стилизация изображений, адаптивность.
- **Favicon**: Иконка сайта в формате ICO.

---

## 📋 Предварительные требования

- **Веб-браузер**: Современный браузер с поддержкой HTML5.
- **Текстовый редактор**: Notepad++, VS Code, Sublime Text или аналог.
- **ОС**: Любая платформа с веб-браузером.

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout html5-css3/anounce-image
```

### 3. Открытие сайта
Откройте www/index.html в браузере.

## 💡 Работа с изображениями

### Добавление изображения
```html
<img src="path/to/image.jpg" alt="Описание изображения">
```

### Добавление Favicon
```html
<link href="img/favicon.ico" rel="shortcut icon" type="image/x-icon">
```

### Адаптивные изображения в CSS
```css
img {
    max-width: 100%;
    height: auto;
}
```
