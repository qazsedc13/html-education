# Урок 6 - Продвинутые теги HTML

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по продвинутым текстовым тегам HTML. Изучение аббревиатур `<abbr>`, цитат `<blockquote>`, `<q>`, кода `<code>`, `<pre>`, индексов `<sup>`, `<sub>`.

---

## 🚀 Основные возможности

- **Аббревиатуры**: Тег `<abbr>` для сокращений
- **Цитаты**: Теги `<blockquote>`, `<cite>`, `<q>`
- **Код**: Теги `<code>`, `<pre>`, `<samp>`, `<kbd>`
- **Индексы**: Теги `<sup>` (верхний) и `<sub>` (нижний)
- **Address**: Тег `<address>` для контактной информации

---

## 🛠 Технологический стек

- **HTML5**: Продвинутые текстовые теги
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout lesson-6/teg-next
```

### 3. Открытие сайта

