# Создание сайта 10 - Основная часть

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию основной части сайта в рамках курса по созданию сайта с нуля. Изучение контент-области, статей, постов и стилизации материала.

---

## 🚀 Основные возможности

- **Контент-область**: Основная часть страницы
- **Статьи/Посты**: Карточки контента
- **Типографика**: Оформление текста
- **Изображения**: Стили для картинок

---

## 🛠 Технологический стек

- **HTML5**: Разметка контента
- **CSS3**: Стилизация статей
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-10/osnovnaya_chast_sajta
```

### 3. Открытие сайта
```bash
cd mySite
start index.html
```
