# Создание сайта 13 - Обратная связь

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)
![CSS3](https://img.shields.io/badge/CSS3-3-blue)

Учебный проект по созданию страницы обратной связи в рамках курса по созданию сайта с нуля. Изучение форм контактов, валидации и стилизации.

---

## 🚀 Основные возможности

- **Форма контактов**: Поля для связи с пользователем
- **Валидация**: Проверка данных на клиенте
- **Стилизация**: Оформление формы
- **Адаптивность**: Форма на мобильных устройствах

---

## 🛠 Технологический стек

- **HTML5**: Форма и поля ввода
- **CSS3**: Стилизация формы
- **JavaScript**: Валидация (опционально)
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code

---

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone git@github.com:qazsedc13/html-education.git
cd html-education
```

### 2. Переключение на ветку
```bash
git checkout sozdanie_sajta-13/obratnaya_svyaz
```

### 3. Открытие сайта
```bash
cd mySite
start index.html
```
