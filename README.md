# Урок 14 - Формы: input

![License](https://img.shields.io/github/license/qazsedc13/html-education)
![HTML5](https://img.shields.io/badge/HTML5-5-orange)

Учебный урок по элементу `<input>`. Изучение текстовых полей, паролей, чекбоксов, радиокнопок, файлов, типов input (text, password, checkbox, radio, file, email, tel, url, number, date).

---

## 🚀 Основные возможности

- **Тег `<input>`**: Создание полей ввода
- **Типы полей**: text, password, checkbox, radio, file
- **HTML5-типы**: email, tel, url, number, date, color, range
- **Атрибуты**: placeholder, required, pattern, min, max
- **Валидация**: Встроенная проверка данных

---

## 🛠 Технологический стек

- **HTML5**: Формы и элементы ввода
- **Веб-браузер**: Chrome, Firefox, Edge, Safari

---

## 📋 Предварительные требования

- **Веб-браузер**: Любой современный браузер
- **Текстовый редактор**: Блокнот или VS Code
